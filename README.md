# Shelf Check

Astro static site for FDA food recall intelligence, AI-assisted editorial content, brand accountability scoring, and GitHub Pages deployment.

## Stack

- Astro 5 (SSG)
- Tailwind CSS 4
- Keystatic CMS
- Chart.js islands
- TypeScript scripts for data collection and content generation
- GitHub Actions + GitHub Pages

## Commands

```bash
npm install
npm run collect
npm run analyze
npm run generate:content
npm run generate:images
npm run build
```

## Admin

Keystatic is configured in GitHub mode for `bullwinkle/ShelfCheck`.
Run `npm run dev` locally and open `/keystatic` during development.
